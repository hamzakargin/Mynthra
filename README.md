## About Mynthra üçÉ

**Mynthra** is a modern dark VSCode theme designed for developers who value clarity and visual harmony in their coding environment. Built with the JavaScript and TypeScript ecosystem in mind, it provides carefully crafted syntax highlighting that enhances code readability without sacrificing aesthetics.

### Two Variants, One Vision üçÇ

- **Mynthra** ‚Äî The classic variant with a deeper, darker background and mint-green accents that keep your code crisp and focused
- **Mynthra Vaporwave** ‚Äî A softer, pale dark variant specifically designed for Angular development with adjusted syntax highlighting

Both variants share the same philosophy: reduce eye strain during extended coding sessions while maintaining excellent syntax distinction for JavaScript, TypeScript, React, Angular, Node.js, HTML, CSS, and JSON.

- **Optimized for JavaScript/TypeScript** ‚Äî Syntax colors designed specifically for modern web development

## Installation

### From VSCode Marketplace (Recommended)

1. Open VSCode
2. Go to Extensions (`Ctrl+Shift+X` or `Cmd+Shift+X`)
3. Search for **"Mynthra"**
4. Click **Install**

### Quick Install via Command Palette

1. Press `Ctrl+P` (Windows/Linux) or `Cmd+P` (macOS)
2. Type: `ext install Mynthra.mynthra-theme`
3. Press Enter

### Manual Installation

Download the latest `.vsix` file from [GitHub Releases](https://github.com/hamzakargin/Mynthra/releases) and install via:
- `Extensions: Install from VSIX...` in the Command Palette

---

## Activation

After installation:

1. Open Command Palette (`Ctrl+Shift+P` or `Cmd+Shift+P`)
2. Type **"Color Theme"** and select **Preferences: Color Theme**
3. Choose either:
   - **Mynthra** (classic dark)
   - **Mynthra Vaporwave** (purple-tinted)
